remove Assignments to Parameters(移除对参数的赋值)

代码对一个参数进行赋值，
以一个临时变量取代该参数的位置。

对于参数赋值这个说法，意思如果把一个名为foo的对象作为参数传给某个函数，那么“对参赋值”意味着改变foo，使它指向别名的一个对象。
如果在foo对象身上进行什么操作。那是没有问题的。

只针对“foo被改指向别一个对象”

做法:
1，建立临时变量，把待处理的参数赋值于它
2，把对原参数的引用，全替换为对临时变量的引用。
3，修改赋值语句，使用改为对新建立之临时变量赋值。
4，编译，测试


重构第一步
必须有单元测试。


重构前:
public int discount(int inputVal, int quantity, int yeartoDate) {
    if (inputVal > 50) {
      inputVal -= 2;
    }
    if (quantity > 100) {
      inputVal -= 1;
    }
    if (yeartoDate > 10000) {
      inputVal -= 4;
    }
    return inputVal;
  }


重构后:
public int discount(int inputVal, int quantity, int yeartoDate) {
    // 以临时变量取代参数赋值
    int result = inputVal;

    if (inputVal > 50) {
      result -= 2;
    }
    if (quantity > 100) {
      result -= 1;
    }
    if (yeartoDate > 10000) {
      result -= 4;
    }
    return result;
  }
