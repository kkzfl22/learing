Replace Array with Object (以对象取代数组)

你有一个数组，其中的元素各自代表不同的东西。
以对象替换据。对于数组中的每个元素，以一个字段来表示

String[] row = new String[3];
row[0]="LivePool";
row[1]="15"

Performance row = new Performance();
row.setName("LivePool");
row.setWins("15");

原因：
数组是一种常见的用以组织数据的结构，不过它应该只用于“以某种顺序容纳一组相似的对象”
有时一组数组容纳了多种不同对象，这会给用户带来麻烦，因为它们很难记住像“数组的第一个元素是人名”这样的约定。
对象就不同了，可以运用字段名称 和函数名称来传达这样的信息，因此你无需记住它。

做法：
1，新建一个类表示数组所拥有的信息，并以其中一个public字段保存原先的数组
2,修改数组中的所有用户，让他们改用新的类实例
3，编译，测试
4，逐一为数组元素添加取值、设值函数。让它们通过函数取用数组内的元素，每次修改后，编译并测试。
5，当所有的数组的直接访问都转布调用访问函数后，将新类中的保存该数组的字段声明为private
6,编译
7，对于数组内的每个元素，在新类中创建一个类型相当的字段，修改该 元素的访问函数，令它们改用上迷的新建字段
8，数组的所有元素都 有了相应的字段后，删除该数组。
